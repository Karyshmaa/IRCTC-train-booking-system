 IRCTC Train Booking System (Work in Progress)

A *backend-focused train booking system* inspired by IRCTC.  
Currently under development.

 Features Implemented
 - User registration & login (basic)
 - Train listing using JSON data (temporary storage)

Tech Stack
- Java  
- Spring Boot  
- JSON (temporary data storage)  

 Planned Features
- Database integration (MySQL/PostgreSQL)  
- Frontend UI for booking and viewing trains  
- Seat availability logic  
- Payment simulation  

 Status
Work in progress. Actively improving.  

 Notes
- This project is currently *backend-only*  
- JSON is used temporarily until database integration is complete  
- Contributions / suggestions welcome!
